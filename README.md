# metabanalyst
Python based metabolomics exploratory analysis covering PCA, heatmaps, loading plots, volcano plots and Venn diagrams applied to four datasets.
Features

Data preprocessing and normalization

Principal Component Analysis (PCA)

PCA loading plots

Heatmaps for metabolite patterns

Volcano plots for differential metabolite analysis

Venn diagrams to compare significant metabolite sets

Jupyter notebook workflow for reproducibility
